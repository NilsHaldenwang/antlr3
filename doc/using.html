Getting Started With ANTLR3 For Ruby

Step 1: Download / Install

Two Ways To Install

Rubygems

% sudo gem install antlr3   # or whatever form of this command appropriate for your system
  • this will install the runtime library and customized ANTLR jar as a ruby gem

Git

% git clone http://github.com/ohboyohboyohboy/antlr3
 
% cd antlr3
 
% ruby setup.rb
 
% rake install
Step 2: Write an ANTLR Grammar
  • For general information on writing grammars, read up on ANTLR’s official documentation
  • For ruby output, the grammar must declare language = Ruby; in its top-level options block
Step 3: Run the ANTLR Tool
% antlr4ruby path/to/your/Grammar.g
  • the antlr4ruby command is just a simple wrapper around the command java org.antlr.Tool. It makes sure the customized ANTLR jar bundled up with this package is in the java class path to get correct output.
  • for information on using the ANTLR tool command, refer to the official documentation
  • assuming ANTLR doesn’t have any serious complaints about your grammar, you should now have a number of ruby output files containing your recognition code
